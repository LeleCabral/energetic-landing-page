<h1>⚡ Energetic+ | Sinta a Energia Tropical</h1>

<h2>Sobre o Projeto</h2>
Este projeto nasceu de um desafio acadêmico na disciplina de Introdução ao Marketing Digital (Publicidade e Propaganda). A proposta era desenvolver a identidade visual e o branding para uma marca de energético de alta performance.

Para elevar o nível do desafio, decidi unir os conceitos de marketing com o desenvolvimento Front-end, transformando o design estático em uma Landing Page interativa e imersiva. O foco foi criar uma experiência onde a interface comunica a refrescância e a energia natural do produto.

<h3>✨ Funcionalidades e Diferenciais</h3>
Interface Interativa: Navegação dinâmica entre os sabores (Pitaya, Melancia e Blueberry).

Animações Fluídas: Uso de CSS e JavaScript para transições de elementos (latas, frutas e textos).

Identidade Visual: Design vibrante que mescla elementos 3D com ilustrações em watercolor para reforçar o conceito "natural".

<h3>🚀 Tecnologias Utilizadas</h3>
O projeto foi construído utilizando o trio fundamental do desenvolvimento web, sem frameworks, para consolidar a lógica de programação:

HTML5: Estruturação semântica e metatags para SEO/Social Media.

CSS3: Estilização avançada, Flexbox, e animações com transition e transform.

JavaScript (ES6+): Manipulação de DOM para troca dinâmica de conteúdos e estados.

Git/GitHub: Versionamento profissional.
